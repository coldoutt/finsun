Финансовые итоги
================

Рабочий адрес программы:
http://127.0.0.1:8780/

Как запускать:
1. Откройте start.bat.
2. Перейдите в браузере на http://127.0.0.1:8780/
3. Пока открыто окно start.bat, программа может сохранять данные в файл.

Где хранятся данные:
- Основной файл данных: finance-data.json
- Он лежит в этой же папке рядом с index.html, app.js и server.js.

Важно:
- Для обычной работы используйте только порт 8780.
- Порты 8795, 8796, 8797 и похожие были временными портами для проверки и больше не нужны.
- Если открыть index.html напрямую без start.bat, браузер не сможет сохранять данные в файл на компьютере.
